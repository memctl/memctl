# Installation

## Requirements

- Node.js 20 or later
- An MCP-compatible AI coding agent

## Install via npx (Recommended)

No installation needed. Just configure your MCP client:

```json
{
  "mcpServers": {
    "memctl": {
      "command": "npx",
      "args": ["@memctl/cli"],
      "env": {
        "MEMCTL_TOKEN": "your-token",
        "MEMCTL_ORG": "your-org",
        "MEMCTL_PROJECT": "your-project"
      }
    }
  }
}
```

## Install Globally

```bash
npm install -g @memctl/cli
```

Then use `memctl` instead of `npx @memctl/cli` in your MCP config.

## Environment Variables

| Variable | Required | Description |
|---|---|---|
| `MEMCTL_TOKEN` | Yes | API token or JWT |
| `MEMCTL_ORG` | Yes | Organization slug |
| `MEMCTL_PROJECT` | Yes | Project slug |
| `MEMCTL_API_URL` | No | Custom API URL (default: `https://memctl.com/api/v1`) |
