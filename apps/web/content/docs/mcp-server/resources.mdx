# MCP Resources

mem/ctl exposes project memory resources and structured agent-context resources.

## Memory resources

## `memory://project/<slug>`

Returns project memories (paginated by server defaults).

## `memory://project/<slug>/<key>`

Returns a single memory by key.

## `memory://capacity`

Returns memory capacity status:

- `used`
- `limit`
- `isFull`
- `usageRatio`

## Agent context resources

## `agent://functionalities`

Returns the full functionality catalog grouped by type, including:

- counts and items per type
- current git branch data
- memory status

## `agent://functionalities/<type>`

Returns detailed items for one functionality type.

Valid types:

- `coding_style`
- `folder_structure`
- `file_map`
- `architecture`
- `workflow`
- `testing`
- `branch_plan`
- `constraints`

## `agent://branch/current`

Returns current git branch details and the saved branch plan memory (if one exists).
