# MCP Server

mem/ctl runs as a local MCP server launched by your agent. It exposes 11 tools with 90+ actions, read-only resources, and communicates over stdio transport.

- [Tools](/docs/mcp-server/tools)  - all 11 tools and their actions
- [Resources](/docs/mcp-server/resources)  - MCP resource URIs
- [Transports](/docs/mcp-server/transports)  - stdio and planned HTTP transport
