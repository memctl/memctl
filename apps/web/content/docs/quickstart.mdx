# Quickstart

## 1. Create an account

Sign up at [memctl.com](https://memctl.com) with GitHub. Create an organization and project in the dashboard.

## 2. Authenticate

```bash
npx memctl auth
```

This prompts for your API token (get it from **Settings â†’ API Tokens** in the dashboard), verifies it, and saves it to `~/.memctl/config.json`. You only do this once per machine.

## 3. Configure your agent

After authenticating, your MCP config only needs the org and project  - no token in project files:

### Claude Code

Create `.mcp.json` in your project root:

```json
{
  "mcpServers": {
    "memctl": {
      "command": "npx",
      "args": ["-y", "memctl@latest"],
      "env": {
        "MEMCTL_ORG": "your-org",
        "MEMCTL_PROJECT": "your-project"
      }
    }
  }
}
```

### Cursor

Create `.cursor/mcp.json` in your project root:

```json
{
  "mcpServers": {
    "memctl": {
      "command": "npx",
      "args": ["-y", "memctl@latest"],
      "env": {
        "MEMCTL_ORG": "your-org",
        "MEMCTL_PROJECT": "your-project"
      }
    }
  }
}
```

### VS Code + GitHub Copilot

Create `.vscode/mcp.json` in your project root:

```json
{
  "servers": {
    "memctl": {
      "type": "stdio",
      "command": "npx",
      "args": ["-y", "memctl@latest"],
      "env": {
        "MEMCTL_ORG": "your-org",
        "MEMCTL_PROJECT": "your-project"
      }
    }
  }
}
```

For other agents, see the [full agent setup guides](/docs/agents).

## 4. Verify

```bash
npx memctl doctor
```

Or ask your agent: *"Use the memory tool with action capacity to check my memory usage."*

## 5. Start using

Your agent now has access to persistent memory. Try:

- *"Store our coding conventions as a coding_style context entry"*
- *"What architecture decisions have been recorded for this project?"*
- *"Create a branch plan for the auth refactor"*

## Alternative: inline token

If you prefer not to run `memctl auth` (CI, shared environments), include the token directly:

```json
{
  "mcpServers": {
    "memctl": {
      "command": "npx",
      "args": ["-y", "memctl@latest"],
      "env": {
        "MEMCTL_TOKEN": "your-token",
        "MEMCTL_ORG": "your-org",
        "MEMCTL_PROJECT": "your-project"
      }
    }
  }
}
```
