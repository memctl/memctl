# API Endpoints

Base URL: `https://memctl.com/api/v1`

All endpoints require authentication via Bearer token.

## Memories

### List Memories

```
GET /memories
```

**Headers:**
- `X-Org-Slug` — Organization slug
- `X-Project-Slug` — Project slug

**Query Parameters:**
- `q` — Search query (optional)
- `limit` — Max results (default: 100, max: 100)
- `offset` — Pagination offset (default: 0)

### Store Memory

```
POST /memories
```

**Body:**

```json
{
  "key": "architecture",
  "content": "We use Next.js with App Router...",
  "metadata": { "category": "decisions" }
}
```

### Get Memory

```
GET /memories/:key
```

### Update Memory

```
PATCH /memories/:key
```

**Body:**

```json
{
  "content": "Updated content...",
  "metadata": { "updated": true }
}
```

### Delete Memory

```
DELETE /memories/:key
```

## Organizations

### List Organizations

```
GET /orgs
```

### Create Organization

```
POST /orgs
```

**Body:**

```json
{
  "name": "My Team",
  "slug": "my-team"
}
```

## Projects

### List Projects

```
GET /projects?org=org-slug
```

### Create Project

```
POST /projects?org=org-slug
```

**Body:**

```json
{
  "name": "My App",
  "slug": "my-app",
  "description": "Optional description"
}
```

## API Tokens

### List Tokens

```
GET /tokens?org=org-slug
```

### Create Token

```
POST /tokens?org=org-slug
```

**Body:**

```json
{
  "name": "Claude Code"
}
```

### Revoke Token

```
DELETE /tokens?id=token-id
```
